## 推荐的 ModelPrice 配置

为了使计算结果与官方价格一致，建议配置以下 ModelPrice（每秒价格）：

```json
{
  "kling-video-o1": 0.6,
  "kling-v1": 0.2,
  "kling-v1-5": 0.4,
  "kling-v1-6": 0.4,
  "kling-v2-1": 0.4,
  "kling-v2-1-master": 2.0,
  "kling-v2-master": 2.0,
  "kling-v2-5-turbo": 0.3,
  "kling-v2-6": 0.5
}
```

### 特殊功能的 ModelPrice

| 功能 | 推荐 ModelPrice | 说明 |

## 当前实现的倍率配置

### proScaleMap (Pro/Std 倍率)

```go
var proScaleMap = map[string]float64{
    "kling-video-o1":   0.8 / 0.6,    // 1.333
    "kling-v2-6":       1.0,          // V2.6 只有 Pro，无 Std
    "kling-v2-5-turbo": 2.5 / 1.5,    // 1.667
    "kling-v2-1":       3.5 / 2.0,    // 1.75
    "kling-v1-6":       3.5 / 2.0,    // 1.75
    "kling-v1-5":       3.5 / 2.0,    // 1.75
    "kling-v1":         3.5 / 1.0,    // 3.5
}
```

### withVideoInputScaleMap (有视频输入倍率)

```go
var withVideoInputScaleMap = map[string]float64{
    "kling-video-o1": 0.9 / 0.6,  // 1.5
```

```go
var soundScaleMap = map[string]float64{
```

## 验证公式

```
最终价格 = ModelPrice × duration × modeScale × advanceScale × groupRatio
```

其中：
- `ModelPrice`: 系统配置的模型基础价格（每秒）
- `duration`: 视频时长（秒）
- `modeScale`: 模式倍率（Std=1.0, Pro=proScaleMap[model]）
- `advanceScale`: 高级功能倍率（视频输入、音频、音色等）
- `groupRatio`: 用户分组倍率
